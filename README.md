# 🎓 Student Registration System
A full-stack web application to manage student records using Flask and SQLite.

## 🚀 Features
- Add Student Records
- View Dashboard
- Search Students
- Edit Details
- Delete Records
- Export CSV
- Real Analytics Chart
- Attractive Responsive UI

## 🛠 Tech Stack
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js

## ▶ How to Run
1. Open project in VS Code
2. Run:
python app.py

3. Open browser:
http://127.0.0.1:5000

- Home Page
- Register Page
